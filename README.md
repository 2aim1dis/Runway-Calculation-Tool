## Getting Started

This is a basic web app template ready for your custom instructions.

### Prerequisites

- Node.js and npm (optional, for development server)
- A modern web browser

### Running the App

**Option 1: Simple file opening**
- Open `index.html` directly in your web browser

**Option 2: Development server (requires Node.js)**
1. Install Node.js from https://nodejs.org/ if not already installed
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm start
   ```
   This will automatically open the app in your browser at http://localhost:8080
